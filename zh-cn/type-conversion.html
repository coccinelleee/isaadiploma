Типті түрлендіру

Типті түрлендіру — бағдарламалауда жиі кездесетін қажеттілік. Python және JavaScript тілдерінде типтерді түрлендіруге арналған арнайы әдістер мен функциялар бар.

Тапсырма​

8192MB көлеміндегі қатқыл диск бар деп есептейік (мән жол ретінде берілген). Осы дискінің көлемін TB (терабайт) бірлігіне түрлендіріп, нәтижені бүтін сан айнымалысында сақтаңыз.

JavaScript іске асыруы​

📄 Python коды
let gb = '8192MB';
let tb = parseInt(gb) / 1024;
let intTb = parseInt(tb);
console.log(`Бұл қатқыл дискінің көлемі: ${intTb}TB`);

Python іске асыруы​

📄 Python коды
gb = '8192MB'
int_tb = int(gb[:-2]) // 1024
print(f"Бұл қатқыл дискінің көлемі: {int_tb}TB")

Кодты түсіндіру​

  • Python-да типті түрлендіру үшін типтің атымен аттас кірістірілген функциялар қолданылады (str(), bool(), int() және т.б.).
  • JavaScript тілінде parseInt, parseFloat, Boolean, String сияқты функциялар қолданылады.
  • Python тілінде '8192MB' сияқты жолды int() арқылы тікелей түрлендіруге болмайды. Алдымен 'MB' бөлігін алып тастау қажет (мысалы, тілім арқылы: [:-2]).
  • JavaScript parseInt() функциясы санмен басталатын жолдан автоматты түрде санды бөліп ала алады.
  • Python жолды тілім (slicing) синтаксисі арқылы өңдей алады: [бастап:соңына_дейін].

Жылдам салыстыру​

МүмкіндікJavaScriptPython
Бүтін санға түрлендіруparseInt(val) / Number(val)int(val)
Нақты санға түрлендіруparseFloat(val) / Number(val)float(val)
Жолға түрлендіруval.toString() / String(val)str(val)
Логикалық мәнгеBoolean(val)bool(val)
Тізімге түрлендіру–list(val)
Сөздікке түрлендіру–dict(val)
