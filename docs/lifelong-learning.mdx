Python бағдарламалау негіздерін меңгергеніңізбен құттықтаймын!

Әрі қарай өтпес бұрын, үйренген біліміңізді тексеруге аздап уақыт бөлейік.

Төменде Python тілінде жүзеге асырылған жасанды интеллектке қатысты бірнеше жоба берілген. Менің ойымша, сіз бұл кітапты оқығаннан кейін олармен жұмыс істеуге толық қабілеттісіз.

- OpenAI Python SDK - https://github.com/openai/openai-python  
- Open Interpreter - https://github.com/KillianLucas/open-interpreter  
- Langchain - https://github.com/langchain-ai/langchain
