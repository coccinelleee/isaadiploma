Python — қарапайымдылығы және оқуға жеңілдігімен танымал, кеңінен қолданылатын бағдарламалау тілі. Python тілінде тиімді жұмыс істеу үшін жергілікті даму ортасын орнату — маңызды қадам.

Python-ның екі негізгі нұсқасы бар: 2.x және 3.x. Python 3.x нұсқасы көптеген жаңа мүмкіндіктерге ие және кеңінен қолданылады, сондықтан бұл кітапта қазіргі таңда ең соңғы Python 3.x нұсқасы — Python 3.12 негізге алынады.

### Python Орнату
Python тілінде бағдарламалау мен жөндеуді бастау үшін интерпретаторды (Python) орнату қажет. Орнату процесі қолданылып отырған операциялық жүйеге байланысты әртүрлі болады.

#### Windows

1. [Python ресми сайтына](https://www.python.org/downloads/) кіріп, Windows үшін соңғы Python орнатқышын жүктеп алыңыз.
2. Орнатқышты іске қосып, нұсқаулықтарға сәйкес орнатыңыз.
3. Орнату барысында **"Add Python to PATH"** параметрін міндетті түрде белгілеңіз. Бұл Python командалық жолда бүкіл жүйеде қолжетімді болуын қамтамасыз етеді.

#### macOS

1. macOS жүйесінде әдетте Python алдын ала орнатылған. Бірақ бұл кітаптағы мысалдармен сәйкестік үшін Python 3.x-тің соңғы нұсқасын орнату ұсынылады.
2. [Python ресми сайтына](https://www.python.org/downloads/) кіріп, macOS үшін соңғы нұсқасын жүктеп алыңыз.
3. Орнатқышты іске қосып, шебердің нұсқауларын орындаңыз.

:::tip 
Егер сіздің жүйеңізде Homebrew орнатылған болса, Python-ды келесі пәрменмен орната аласыз: brew install python@3
:::

### Python Нұсқасын Тексеру
Орнату аяқталғаннан кейін терминалда келесі пәрменді орындаңыз:

<PythonToggleCodeBlock
  code={`python —version
  
немесе:
  
python -v`}
  language="bash"
  title="Python нұсқасын тексеру"
/>

:::tip `python` пәрмені табылмады ма?
Python пәрмені жұмыс істемесе, төмендегілерді тексеріңіз:

1. `$PATH` айнымалысында Python орнатылған қалта бар ма?
2. `python3 --version` пәрменін орындап көріңіз.

Егер тек `python3` жұмыс істесе, онда мынадай бүркеншік ат (alias) орнатуға болады:

<CodePythonBlock
  title="Alias орнату"
  language="bash"
  code={`alias python=python3`}
/>
:::

### Интеграцияланған Даму Ортасы (IDE)

Python-да өнімді жұмыс істеу үшін интеграцияланған даму ортасын (IDE) пайдалану өте маңызды. Төменде Python үшін кеңінен қолданылатын үш танымал IDE берілген:

1. **Visual Studio Code (VS Code)**  
   VS Code — Microsoft әзірлеген, жеңіл, функционалға бай және кеңейтілетін IDE. Ол ашық бастапқы кодты, көп платформалы және Python тілімен тамаша интеграцияланады. [Осы жерден жүктеуге болады](https://code.visualstudio.com/). Код жазу кезінде автотолтыру мүмкіндігін пайдалану үшін [Python кеңейтімін](https://marketplace.visualstudio.com/items?itemName=ms-python.python) орнату қажет.

2. **PyCharm Community Edition**  
   PyCharm — Python бағдарламалау үшін арнайы жасалған қуатты IDE. Community Edition нұсқасы тегін, ашық бастапқы кодты және автотолтыру, жөндеу, нұсқа бақылауы секілді толық функцияларды ұсынады. [Мұнда жүктей аласыз](https://www.jetbrains.com/pycharm/download/).

3. **Spyder**  
   Spyder — ғылыми Python жобалары үшін кеңінен қолданылатын ашық бастапқы кодты IDE. Ол деректерді талдау, графиктер құру және жөндеу мүмкіндіктерін біріктіреді. [Spyder ресми сайтынан жүктеп алуға болады](https://www.spyder-ide.org/).
