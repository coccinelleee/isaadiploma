Модульдерді құру және импорттау

Үлкен көлемді қосымшалар әдетте көптеген модульдерден тұрады. Модуль — бұл бағдарламалық жасақтамадағы тәуелсіз код бірлігі. Python тілінде әрбір .py файлы — модуль, ал JavaScript тілінде әрбір .js файлы — модуль.

Тапсырма​

Қарапайым арифметикалық амалдарды (қосу, азайту, көбейту, бөлу) орындайтын калькулятор модулін жасаңыз. Бұл модульді басқа модульге импорттап, 1234 * 4321 амалын орындап, нәтижесін басып шығарыңыз.

JavaScript іске асыруы​

📄 calculator.js
export function add(a, b) {
  return a + b;
}

export function subtract(a, b) {
  return a - b;
}

export function multiply(a, b) {
  return a * b;
}

export function divide(a, b) {
  if (b === 0) {
    throw new Error("Нөлге бөлуге болмайды");
  }
  return a / b;
}
📄 main.js
import { multiply } from './calculator.js';
console.log(multiply(1234, 4321));

Python іске асыруы​

📄 calculator.py
def add(a, b):
    return a + b

def subtract(a, b):
    return a - b

def multiply(a, b):
    return a * b

def divide(a, b):
    if b == 0:
        raise ValueError("Нөлге бөлуге болмайды")
    return a / b
📄 main.py
import calculator
result = calculator.multiply(1234, 4321)
print(result)

Кодты түсіндіру​

  • Python және JavaScript тілдерінде де модульдерді импорттау үшін import кілт сөзі қолданылады.
  • Python тілінде модуль мүшелерін арнайы export кілт сөзімен жариялау қажет емес — барлық мүшелер әдепкі бойынша қолжетімді болады.
Python модульдеріндегі жеке (private) мүшелер

Python-да модуль ішіндегі барлық айнымалылар, функциялар және кластар әдепкі бойынша экспортталады және басқа модульдерден импортталуы мүмкін. Python жеке мүшелерді қорғаудың арнайы механизмін ұсынбайды. Дегенмен, белгілі бір мүшелерді "жеке" ретінде белгілеу үшін олардың атауын _ немесе __ таңбасымен бастау — қабылданған тәжірибе. Үшінші тарап модульдерін қолданғанда осындай мүшелерді импорттамаған жөн.

Python тілінде модуль атауы файл атауымен бірдей болады. Файл атауында - таңбасы болмауы керек, әйтпесе модульді импорттау мүмкін болмайды.

Python файл атаулары мен модуль атаулары арасында келесі ережелерді ұстанады:
