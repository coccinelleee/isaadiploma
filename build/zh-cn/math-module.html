Math модулі

Python тіліндегі math модулі — әртүрлі математикалық функциялар мен тұрақтыларды ұсынатын кітапхана.

Тапсырма​

Берілген радиус бойынша шеңбердің ауданын есептейтін бағдарлама жазыңыз.

JavaScript іске асыруы​

📄 Python коды
const radius = 5;
const area = Math.PI * Math.pow(radius, 2);
console.log(area);

Python іске асыруы​

📄 Python коды
import math

radius = 5
area = math.pi * math.pow(radius, 2)
print(area)
