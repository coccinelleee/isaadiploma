Негізгі деректер типтерімен арифметикалық амалдар

Python тіліндегі арифметикалық операторлар қосу, азайту, көбейту және бөлу сияқты негізгі арифметикалық амалдарды орындауға арналған.

Тапсырма​

Екі сандық айнымалыны анықтайтын және келесі арифметикалық амалдарды орындайтын бағдарлама құрыңыз:

  1. Екі санды қосып, нәтижесін басып шығару.
  2. Бірінші саннан екінші санды азайтып, нәтижесін басып шығару.
  3. Екі санды көбейтіп, нәтижесін басып шығару.
  4. Бірінші санды екінші санға бөліп, нәтижесін басып шығару.
  5. Бірінші санды екінші санға бөлгендегі қалдықты есептеп, басып шығару.
  6. Бірінші санды екінші санның дәрежесіне шығарып, нәтижесін басып шығару.
  7. Бірінші санды 1-ге арттырып, жаңартылған мәнін басып шығару.
  8. Екінші санды 1-ге кемітіп, жаңартылған мәнін басып шығару.

JavaScript іске асыруы​

📄 JavaScript
let num1 = 10;
let num2 = 5;

// 1. Қосу
let sum = num1 + num2;
console.log("Қосу нәтижесі:", sum);

// 2. Азайту
let difference = num1 - num2;
console.log("Айырма:", difference);

// 3. Көбейту
let product = num1 * num2;
console.log("Көбейтінді:", product);

// 4. Бөлу
let quotient = num1 / num2;
console.log("Бөлінді:", quotient);

// 5. Қалдық
let remainder = num1 % num2;
console.log("Қалдық:", remainder);

// 6. Дәреже
let power = num1 ** num2;
console.log("Дәреже:", power);

// 7. Арттыру
num1++;
console.log("num1 мәні арттырылғаннан кейін:", num1);

// 8. Кеміту
num2--;
console.log("num2 мәні кемітілгеннен кейін:", num2);

Python іске асыруы​

📄 Python
num1 = 10
num2 = 5

# 1. Қосу
sum_val = num1 + num2
print("Қосу нәтижесі:", sum_val)

# 2. Азайту
difference = num1 - num2
print("Айырма:", difference)

# 3. Көбейту
product = num1 * num2
print("Көбейтінді:", product)

# 4. Бөлу
quotient = num1 / num2
print("Бөлінді:", quotient)

# 5. Қалдық
remainder = num1 % num2
print("Қалдық:", remainder)

# 6. Дәреже
power = num1 ** num2
print("Дәреже:", power)

# 7. Арттыру
num1 += 1
print("num1 мәні арттырылғаннан кейін:", num1)

# 8. Кеміту
num2 -= 1
print("num2 мәні кемітілгеннен кейін:", num2)

Кодты түсіндіру​

  • Python және JavaScript тілдеріндегі қосу, азайту, көбейту, бөлу, қалдық, дәреже операторларының синтаксисі ұқсас.
  • sum — Python тілінде кіріктірілген функция, сондықтан оны айнымалы атауы ретінде қолданудан сақ болу керек.
  • Python тілінде ++ және -- операторлары жоқ. Оның орнына += және -= қолданылады.

危险

Python тілінде арифметикалық амалдар тек сәйкес келетін деректер типтері арасында ғана орындалуы мүмкін. Мысалы, нақты сан мен бүтін сан арасындағы есептеудің нәтижесі нақты сан болады. Бірақ жол (string) мен сан (number) арасында амал орындау қате тудырады. Мұндайда, біреуін екіншісіне сәйкес типке түрлендіру қажет.
