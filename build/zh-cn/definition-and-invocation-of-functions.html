Функцияларды анықтау және шақыру

Функциялар код логикасын инкапсуляциялау және оқшаулау үшін қолданылады. Python және JavaScript тілдерінде функциялар бірінші дәрежелі объектілер болып саналады.

Тапсырма​

Адамды аты-жөні, жасы және жынысы бойынша сәлемдейтін функция жазыңыз.

JavaScript іске асыруы​

📄 Python коды
function greet(name, age, gender='Құпия') {
  console.log(`Сәлем, менің атым ${name}, жасым ${age}, жынысым: ${gender}`)
}

greet('Боб', 18);
greet('Том', 18, 'ер');
greet(...['Айым', 18, 'әйел']);

Python іске асыру​

📄 Python коды
def greet(name, age, gender='Secret'):
    print(f"Hello, I am {name}, {age} years old, {gender}")


greet('Bob', 18) 
greet(age=18, name='Bob')  
greet('Tom', 18, 'male')
greet(*['Amy', 18, 'female'])

Кодты түсіндіру:​

Python тілінде функциялар def кілт сөзімен анықталады, ал JavaScript тілінде function кілт сөзі пайдаланылады.

Python тілінде функция денесі қос нүктемен : басталып, шегініспен жазылады, ал JavaScript тілінде фигуралы жақшалар ішінде жазылады.

Python тілінде әдепкі мәні жоқ әрбір параметр міндетті. Егер қажет параметр берілмесе, синтаксистік қате пайда болады. JavaScript тілінде берілмеген параметрлерге undefined мәні автоматты түрде беріледі.

Python-да параметрлерді анықталған ретпен беру позициялық аргументтер деп аталады, және олар дәл сол ретпен берілуі тиіс. Сонымен қатар, кілт сөз аргументтері аты=мәні түрінде берілуі мүмкін. Олардың ретін сақтау қажет емес. Позициялық және кілт сөз аргументтерін бірге қолдануға болады, бірақ функцияны шақырғанда барлық қажетті аргументтер берілуі тиіс.

Python-да позициялық аргументтерді менің_функциям(*[а1, а2, ...]) түрінде беруге болады, бұл JavaScript-тегі myFn(...[a1, a2, ...]) сияқты.

Жылдам салыстыру​

ЕрекшелікJavaScriptPython
Функцияны анықтауfunction myFn(params) { ... }def my_fn(params):
Функцияны шақыруmyFn(params)my_fn(params)
Функция аргументтерін қалай алуға болады?

*args — функцияға берілген барлық позициялық аргументтерді алу үшін қолданылады. Олар tuple ретінде сақталады.
**kwargs — барлық кілт сөз аргументтерін алу үшін қолданылады. Олар dict ретінде сақталады.

📄 Python коды
def bir_funktsiya(birinshi_arg, *args, **kwargs):
    print(birinshi_arg)
    print(args)
    print(kwargs)

bir_funktsiya('позициялық_мән','қосымша1','қосымша2',a=1,b=2,c=3)
# Нәтиже:
# позициялық_мән
# ('қосымша1', 'қосымша2')
# {'a': 1, 'b': 2, 'c': 3}
📄 Python коды
def funksiya(a, b, c):
    print(a, b, c)  # 1 2 3

men_dict = {"c": 3, "b": 2, "a": 1}
funksiya(**men_dict)
提示

*args және **kwargs ыңғайлы болғанымен, егер функция тек осылармен шектелсе, онда IDE автоматты кеңестері жоғалады және функция логикасы күрделене түседі. Сондықтан оларды сақтықпен қолданыңыз.
