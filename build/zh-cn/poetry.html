Poetry

Poetry — Python жобалары үшін тәуелділіктерді басқаруды жеңілдететін қуатты құрал.

Poetry орнату​

Poetry-ді қолданбас бұрын, оны операциялық жүйеге орнату қажет. Орнату нұсқаулығы ресми сайтта берілген:
👉 Poetry орнату құжаты

Жаңа жоба жасау​

Poetry орнатылғаннан кейін, жаңа жобаны келесі командамен жасауға болады:

📄 Python коды
poetry new myproject

Бұл команда myproject атты жаңа жоба қалтасын жасайды және оның түбірінде pyproject.toml атты файл пайда болады. Бұл файл JavaScript жобаларындағы package.json файлына ұқсас.

Мысал конфигурациясы:

📄 Python коды
[tool.poetry]
name = "myproject"
version = "0.1.0"
description = ""
authors = ["luckrnx09 <luckrnx09@gmail.com>"]
readme = "README.md"

[tool.poetry.dependencies]
python = "^3.11"

[build-system]
requires = ["poetry-core"]
build-backend = "poetry.core.masonry.api"

Үшінші тарап пакеттерін басқару​

Poetry қолданылған жобаларда pip қолданудың қажеті жоқ. Оның орнына:

📦 Пакетті орнату:

📄 Python коды
poetry add package_name

❌ Пакетті жою:

📄 Python коды
poetry remove package_name

Бұл әрекеттер pyproject.toml файлын автоматты түрде жаңартады, сондықтан pip freeze арқылы тәуелділіктерді қолмен сақтау қажет емес.

Егер жоба қалтасында pyproject.toml файлы бар болса, тек мына команданы орындау жеткілікті:

📄 Python коды
poetry install

Python скрипттерін іске қосу​

Poetry арқылы басқарылатын жобаларда Python скрипттерін іске қосу үшін келесі команданы пайдаланыңыз:

📄 Python коды
poetry run python your_script.py

Бұл Poetry ортасындағы тәуелділіктерді дұрыс жүктеуге көмектеседі.
