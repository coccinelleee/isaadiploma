Класс және объект

Кластар мен объектілер қайта пайдалануға болатын кодты құру және ұйымдастыру үшін қолданылады. Класс — бұл объектілерді құруға арналған сызба, ал объект — сол кластың нақты бір данасы.

Тапсырма​

Шеңбердің ауданын және шеңбер ұзындығын есептей алатын Circle деп аталатын класс жазыңыз.

JavaScript іске асыруы​

📄 JavaScript — Circle class
class Circle {
  static pi = Math.PI
  constructor(radius) {
    this.radius = radius;
  }

  static createCircle(radius){
    return new Circle(radius)
  }

  getArea() {
    return Circle.pi * this.radius ** 2;
  }

  getCircumference() {
    return 2 * Circle.pi * this.radius;
  }

  toString() {
    return `[Circle]: radius:${this.radius}`
  }
}

const circle = new Circle(5); // немесе Circle.createCircle(5)
console.log(circle.toString()); // [Circle]: radius:5
console.log(circle.getArea()); // 78.53981633974483
console.log(circle.getCircumference()); // 31.41592653589793
кеңес

Сыныптардағы static кілт сөзі TypeScript-те бұрыннан бар, бірақ ES2022 бастап JavaScript-те де қолдау табады. Сонымен қатар, TypeScript public, private, және protected сияқты қатынас модификаторларын ұсынады.

Python іске асыруы​

📄 Python — Circle class
import math

class Circle:
  pi = math.pi
  def __init__(self, radius):
    self.radius = radius

  @classmethod
  def create_circle(cls, radius):
    return Circle(radius)
  
  def get_area(self):
    return Circle.pi * self.radius ** 2

  def get_circumference(self):
    return 2 * Circle.pi * self.radius

  def __str__(self):
    return f"[Circle] radius:{self.radius}"

circle = Circle(5) # немесе Circle.create_circle(5)
print(circle) # [Circle] radius:5
print(circle.get_area()) # 78.53981633974483
print(circle.get_circumference()) # 31.41592653589793

Кодты түсіндіру​

  • Python-да объект жасау үшін класс атауын тікелей шақыру жеткілікті (Circle(5)), ал JavaScript-те new кілт сөзі қажет (new Circle(5)).
  • Python-да __init__ арнайы әдісі конструктор ретінде қолданылады, ал JavaScript-те constructor.
  • Python-дағы әдістердің алғашқы аргументі әрқашан self деп аталып, объектінің өзіне сілтеме жасайды.
  • @classmethod арқылы белгіленген әдістер бірінші аргумент ретінде cls алады — бұл классқа сілтеме.
  • Python-да шегініс (indentation) арқылы блоктар анықталады, ал JavaScript {} бұйра жақшаларды қолданады.
  • _ немесе __ арқылы басталатын мүшелер Python-да жеке (private) немесе арнайы (special) деп есептеледі.
  • TypeScript-те мұндай модификаторлар (private, protected) нақты синтаксиспен көрсетіледі.
Python-дағы арнайы әдістер

Python класстарында жиі қолданылатын арнайы әдістер бар. Оларды қайта жазу арқылы объектілерді әртүрлі жағдайларда икемді етуге болады:

  • __init__: Объектіні инициализациялау.
  • __str__: print() қолданғанда қандай мәтін шығатынын анықтайды.
  • __repr__: Объектіні интерпретаторда басып шығарғанда қолданылатын көрініс.
  • __len__: len(obj) арқылы ұзындығын қайтару.
  • __getitem__: obj[index] арқылы элементке қол жеткізу.
  • __setitem__: obj[index] = value арқылы мән орнату.
  • __delitem__: del obj[index] арқылы элементті жою.
  • __getattr__: Объект ішінде жоқ атрибутқа жүгінгенде шақырылады.
  • __setattr__: Атрибут мәнін орнату кезінде жұмыс істейді.
  • __delattr__: Атрибутты del арқылы жою кезінде жұмыс істейді.
