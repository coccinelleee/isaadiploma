OS модулі

Python тіліндегі os модулі файлдар мен бумаларды басқаруға және жүйеге байланысты ақпараттарға қол жеткізуге арналған функцияларды ұсынады.

Тапсырма​

Берілген буманың ішіндегі барлық файлдар мен бумаларды тізіп шығатын бағдарлама жазыңыз. Әр элементтің атауын басып шығарып, ол файл ма, әлде бума ма екенін көрсетіңіз.

JavaScript іске асыруы​

📄 Python коды
import os

def list_items(directory):
    items = os.listdir(directory)

    for item in items:
        item_path = os.path.join(directory, item)
        if os.path.isdir(item_path):
            print(f"{item}: бума")
        else:
            print(f"{item}: файл")
list_items('../')
