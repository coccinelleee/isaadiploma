Негізгі деректер типтері

Python тіліндегі негізгі деректер типтеріне мыналар жатады:

  • Integer (бүтін сан): Бүтін сандар, ондық нүктесіз.
  • Float (нақты сан): Ондық нүктесі бар сандар.
  • String (жол): Бір немесе қос тырнақшаға алынған символдар тізбегі.
  • Boolean (логикалық): True немесе False мәндері.
  • None: Бос мәнді білдіреді.

Тапсырма​

Адамның жалақы парағын сипаттайтын 5 айнымалыны анықтаңыз: аты, айы, жалақысы, мінсіз қатысымы бар ма, және жобалық тобы.

JavaScript іске асыруы​

📄 JavaScript
let name = 'Bob' // жол (string)
let month = 12 // сан (number)
let salary = 10000.12 // сан (number)
let isPerfectAttendance = true // логикалық (boolean)
let team = null // бос мән (null)
console.log(typeof name) // Айнымалының типін анықтау

Python іске асыруы​

📄 Python
name = 'Bob' # str (жол)
month = 12 # int (бүтін сан)
salary = 10000.12 # float (нақты сан)
is_perfect_attendance = True # bool (логикалық)
team = None # None (бос мән)
print(type(name)) # Айнымалының типін анықтау

Кодты түсіндіру​

  • Python тілінде int — бүтін сандар үшін, float — ондық сандар үшін, ал JavaScript екі түрге де ортақ number типін қолданады.
  • Python None бос мән ретінде қолданады, ал JavaScript — null.
  • Python тілінде undefined сияқты ұғым жоқ.
  • Python айнымалының типін білу үшін type() функциясын, ал JavaScript — typeof операторын қолданады.
