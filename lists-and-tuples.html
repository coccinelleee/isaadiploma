Тізімдер мен Кортеждер

Python тілінде массивтер (arrays) тізімдер (lists) деп аталады және бірнеше мәнді тізбектей сақтау үшін қолданылады.

Тапсырма​

Bob, Tom және Amy есімді адамдар тізімін жасаңыз.

JavaScript іске асыруы​

📄 Python коды
let names = [];
names.push('Bob');
names.push('Tom');
names.push('Amy');
console.log(names);

Python іске асыруы​

📄 Python коды
names = []
names.append('Bob')
names.append('Tom')
names.append('Amy')
print(names)

Кодты түсіндіру​

  • Python тілінде элемент қосу үшін append() әдісі қолданылады: my_list.append(el)
  • JavaScript тілінде бұл үшін push() қолданылады: myArr.push(el)

Кортеж деген не?

Кортеж (tuple) — өзгертілмейтін (тұрақты) тізім сияқты. Ол жасалған кезде элементтер саны мен мәндері бекітіледі және кейін өзгертілмейді.
